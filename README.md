
# ✋ Hand Gesture Controller

A real-time hand gesture recognition system built using OpenCV, MediaPipe, and Tkinter.

## 🚀 Features

* Real-time hand tracking
* Finger counting
* Gesture recognition:

  * Fist
  * Open Hand
  * One
  * Peace ✌️
  * Rock 🤘
* FPS counter
* Smooth gesture detection
* GUI interface with Start/Stop controls

## 📥 Model File

The app automatically downloads the MediaPipe model on first run.

If needed manually:
